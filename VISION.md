# Visão do Projeto

[English Version](/VISION_EN.md)

Este documento descreve **o porquê, o para quem e o como** deste projeto.
Ele não é um manual técnico, mas um guia de alinhamento para pessoas usuárias,
colaboradoras e mantenedoras.
